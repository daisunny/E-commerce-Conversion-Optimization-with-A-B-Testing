# E-commerce-Conversion-Optimization-with-A-B-Testing
在电商场景中，用户从浏览到购买存在多层流失。本项目基于用户行为数据，通过漏斗分析定位转化瓶颈，并设计AB测试优化策略，从而提升整体转化率。

In the e-commerce scenario, users experience multi-stage attrition from browsing to purchase. Based on user behavior data, this project adopts funnel analysis to identify conversion bottlenecks and designs A/B testing optimization strategies, so as to improve the overall conversion rate.
